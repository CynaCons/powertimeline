Chronochart Application Debug

Test 1: App Loading

The application should load below. If it shows a blank page, the app is broken.

Test 2: Console Inspection

Open browser developer tools and check console for errors:

  • F12 to open dev tools
  • Go to Console tab
  • Look for red error messages
  • Look for failed network requests in Network tab

Expected: No console errors, app renders with sidebar and timeline area

Test 3: Manual Check

If app loads, try these steps:

  1. Click "Toggle developer options" (build icon at bottom left)
  2. Click "Developer Panel" (settings icon)
  3. Click "Napoleon 1769-1821" to load test data
  4. Close dev panel
  5. Check if cards and timeline scales are visible

Expected: Cards should appear, timeline axis should show year labels distributed across width
